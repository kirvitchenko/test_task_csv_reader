## Описание

Скрипт анализирует CSV-файлы с рейтингами и формирует отчёт по брендам.

### Добавление нового отчёта

Чтобы добавить новый отчёт, нужно:

В функции get_report добавить ещё одно условие для обработки нового типа отчёта
(например, как в моей реализации — average, можно добавить min, max, sum и т.д.).

Указать тип отчёта в параметре choices при добавлении аргумента --report в argparse.

При запуске указать отчёт в формате <тип>-<поле>.

### Пример вывода 
```bash
+----+----------+-----------------+
| №  | Бренд    | Средний рейтинг |
+====+==========+=================+
| 1  | apple    | 4.85            |
| 2  | samsung  | 4.73            |
| 3  | xiaomi   | 4.55            |
+----+----------+-----------------+
```
### Пример запуска

```bash
python main.py --files products1.csv products2.csv --report average-rating
