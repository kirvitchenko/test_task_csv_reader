## Описание
Скрипт анализирует CSV-файлы с рейтингами и формирует отчёт по брендам.

### Пример запуска
python main.py --files products1.csv products2.csv --report average-rating

### Добавление нового отчета
Для того чтобы добавить новый отчет необходимо в фунции get_report в ветвлении добавить еще один условный оперетор 
который обрабатывает тип отчета(например, как в моей реализации - average, можно применить min, max, sum и т.д), 
а для того чтобы обработать какое-другое поле, просто нужно указать его после "-" при передаче агрумента во флаг 
"--report", а так же указать тип отчета в формате "<тип>-<поле>" в параметре "choices" при добавлении аргумента в 
"--result"
